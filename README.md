## Tech Stack

### FrontEnd
- React JS
- React Context API + useReducer
- React Router v6
- Vanilla CSS integrated with [Libre UI](https://libreui.netlify.app/) Component Library

### BackEnd
- MockBee

## **Screens -**

- Homepage
- Video explore page
- Filter Videos based on categories
- Individual video page
- Like videos  
- Watch Later Page
- History of seen videos
- Create playlist
- Authentication:
  - User Signup
  - User Login
- 404 Error Page
